Consolidation Notes (for AC phase)

Configure Attack Effect
- Select Affliction Effect / Resolve Affliction Resistance Check / Recover from Affliction: cluster covers acquisition, resolution, recovery. AC must cover condition set definition, attack check prerequisite, resistance type choice, degree of failure mapping, end-of-turn recovery, and third-degree special recovery.
- Select Blast Effect: Damage + Ranged extra preconfiguration. AC confirms 2 pp/rank and ranged attack check vs Dodge.
- Resolve Nullify Opposed Check: AC specifies success branch (effect countered) and failure branch (continues).

Configure Defense Effect
- Configure Partial Immunity: half-rank cost for half effect protection.
- Recover from Death via Immortality: time-rank formula (19 - Immortality rank); rank 20 edge case.

Configure Mobility Effect
- Select Movement Effect and Choose Special Movement Mode: Each mode has sub-rules. AC phase should expand per mode for distinct behaviors.

Configure Control Effect
- Select Create / Enforce / Dissolve cluster: AC must specify permanence option (spend PP = rank).
- Select Summon / Apply Dazed / Dismiss: AC covers PP = rank x 15, PL <= rank, dazed on arrival, directing = move action.

Configure General Effect
- Select Insubstantial: 4 forms, AC confirms cost per form and rank 4 Incorporeal interaction.
- Select Variable: pool = rank x 5, reallocation standard action, PL limits apply, pool resets when unmaintained.

Apply Per-Rank Modifiers
- Enforce Minimum One Point per Rank Floor: fractional cost progression when flaws reduce below 1 pp/rank.

Assign Power Descriptors
- Descriptor matching is flexible; GM is final arbitrator. AC must note flexibility.

Organize Power Arrays
- Add Alternate Effect: total cost (not just rank) must be <= primary. Permanent effects cannot be alternates.
- Reallocate Dynamic Effects: once per turn, free action; cannot exceed pool total.
